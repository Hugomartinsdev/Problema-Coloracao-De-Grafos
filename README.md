# Trabalho de analise e projeto de algoritimo
Feito por mim para o projeto de bimestre da materiaanalise e projeto de algoritimo.
## Objetivo
Cria um algoritmo que tivesse uma resposta aproximada do problema da Coloração de Grafos.
## Solução
Foi criado um codigo em Python com o metodo wesh-powell para parcialmente solucionar esse problema, além de mostrar a taxa de aproximação do meu algoritimo com a solução otima,Sendo essas duas usadas para a taxa de solução, o algoritimo foi feito puramente com python
## Bibiotecas usadas
as bibliotecas usadas para fazer o calcullo dee aproximação foram: 
<ul>
      <li>itertools</li>
      <li>time</li>
</ul>
Para mostrar o grafo foram usadas as bibliotecas:
<ul>
      <li>matplotlib.pyplot</li>
      <li>networkx</li>
</ul>
